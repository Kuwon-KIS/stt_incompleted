# 배치 0개 파일 날짜 기본 숨김 및 토글 계획

작성일: 2026-03-20
범위: 파일 수가 0인 날짜(폴더만 존재)를 기본으로 숨기고, 사용자 토글로 표시 가능하게 개선

## 정책
- 기본 동작: 0개 파일 날짜 숨김
- 사용자 선택: 토글 ON 시 0개 파일 날짜 포함 표시

## 목적
- 노이즈를 줄여 날짜 선택과 분석 대상을 명확히 한다.
- 대량 빈 날짜 구간에서 불필요한 표시/분석을 최소화한다.

## 설계

### API
- date-range, batch-analysis에 include_empty 파라미터를 도입한다.
- 기본값은 false로 둔다.

### UI
- 배치 처리 탭에 "0개 파일 날짜 포함" 토글을 추가한다.
- 토글 변경 시 날짜 목록과 분석 결과를 즉시 재조회한다.

### 데이터 처리
- 우선 date_status.total_files 기준으로 필터링한다.
- 통계 미집계 날짜는 정책적으로 제외하고, 필요 시 별도 동기화 경로를 둔다.

## 대상 파일
- app/static/index.html
- app/static/js/app.js
- app/static/js/api.js
- app/routes/admin.py
- app/database/manager.py

## 검증
- 기본 상태에서 0개 날짜가 목록/상세 테이블에 나타나지 않는지 확인
- 토글 ON 시 0개 날짜가 다시 나타나는지 확인
- 토글 상태별 batch-analysis 대상 날짜 수가 일관적인지 확인
- 대량 빈 폴더 케이스에서 응답 시간과 렌더 성능 개선 확인

## 완료 기준
- 기본 숨김 정책이 UI/API 전 구간에 일관 적용
- 토글로 즉시 전환 가능
- 기존 배치 실행 기능 회귀 없음
